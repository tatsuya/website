<h1 align="center">website</h1>

Source code for [tatsuya.website](https://tatsuya.website), built with [Gatsby](https://www.gatsbyjs.com) and deployed to [Netlify](https://www.netlify.com).

## Development

```shell
npm install
npm run develop
```

The site runs at `http://localhost:8000`.

## Build

```shell
npm run build
npm run serve
```
